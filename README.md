# Time_tracker

## Description
Time\_tracker is a roleplaying game tool to help Dungeon Masters / Story Tellers you keep track of a bunch of in-game stuff. Note that Time\_tracker is (at least for now) specific to the Forgotten Realms setting for Dungeons and Dragons. It tracks dates according to the Calendar of Harptos and illustrates the phases of SelÃ»ne.

## Usage
This is a Library Add On for MapTool. You will need to download and run MapTool first: https://www.rptools.net/toolbox/maptool/

## Installation
* Drag and drop the .mtlib file from your file explorer onto the MapTool map.
or
* Use File -> Add On Libraries... -> Add to add the .mtlib file.
If installing for the first time in a campaign, follow the instructions in the Read Me which should pop up.
